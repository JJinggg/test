# 이치방쿠지 난수 생성 규칙 추정

> 추정일: 2026-03-19
> 근거 데이터: 22세트, A등급 42개, B등급 45개
> 결론: 순수 난수가 아닌 **제약 조건 기반 가중 랜덤 배치**

---

## 1. 추정 규칙 요약

### 규칙 1: 등급별 개수 고정

각 제품마다 등급별 개수가 사전에 정해져 있음:

| 등급 | 가장 흔한 개수 | 비율 |
|------|-------------|------|
| A | 2개 | 72.7% |
| B | 2개 | 77.3% |
| C | 1~2개 | - |
| D 이하 | 제품별 상이 | - |

---

### 규칙 2: A와 B는 "반대편 배치" (분산 규칙)

```
A등급 → 앞쪽 절반에 편향 배치 (31.0%가 처음 20%에)
B등급 → 뒤쪽 절반에 편향 배치 (31.1%가 마지막 20%에)
```

**설계 의도 추정 (고객 만족도 설계):**
- 초반에 뽑는 사람 → A등급 기대감
- 후반에 뽑는 사람 → B등급 기대감
- 어느 시점에 뽑아도 상위 경품 가능성을 느끼게 함

**추정 구간 가중치:**

| 구간 | A등급 가중치 | B등급 가중치 |
|------|-----------|-----------|
| 구간 1 (처음 ~20%) | **30** | 13 |
| 구간 2 (20~40%) | **22** | **27** |
| 구간 3 (40~60%) | 20 | 16 |
| 구간 4 (60~80%) | 12 | 13 |
| 구간 5 (마지막 ~20%) | 16 | **31** |

---

### 규칙 3: 열(Column) 가중치 — "가장자리 선호, 특정 열 기피"

**A등급 열 가중치:**

```
열 1 (좌측):   ██████████████████  19%
열 2:          ██████████████████  19%
열 3 (중앙):   ██████████████████████████  26%  ← 선호
열 4:          ███████  7%                      ← 기피
열 5 (우측):   █████████████████████████████  29%  ← 최선호
```

**B등급 열 가중치:**

```
열 1 (좌측):   ████████████████████  20%
열 2:          ████████████████████████  24%
열 3 (중앙):   █████████  9%                    ← 기피
열 4:          ████████████████████  20%
열 5 (우측):   ███████████████████████████  27%  ← 최선호
```

**추정 이유:**
- **열 5(우측 끝)**: A·B 모두 선호 → 시각적으로 "구석"이라 소비자가 덜 고르는 위치에 고의 배치
- **열 3(중앙)**: A만 선호 → 가장 눈에 띄는 위치에 최상위를 놓아 "뽑힐 것 같은 느낌" 유도
- **열 4**: A 기피 → 열5 옆이라 소비자가 "우측" 느낌으로 뽑을 수 있는데, 여기엔 안 놓음
- **열 3 B 기피**: A가 중앙에 있으니 B는 중앙을 피해 분산

---

### 규칙 4: 상위 등급 간 최소 간격 (Anti-clustering)

A와 B가 바로 인접한 경우가 거의 없음:

```
80-7: A=28,30 / B=45,74  → 최소 간격 15
80-5: A=5,22  / B=27,57  → 최소 간격 5
80-6: A=4,73  / B=24,32  → 최소 간격 20
80-1: A=14,66 / B=11,40  → 최소 간격 3 (가장 가까운 예외)
```

**추정 규칙:**
```
- A와 B 사이 최소 간격 >= 3 (같은 행에 A와 B가 동시 출현하지 않음)
- A끼리도 최소 간격 >= 5 이상 유지 (대부분)
- 같은 행(5칸)에 상위 등급 2개가 동시에 오는 경우는 극히 드묾
```

---

### 규칙 5: 뽑기 수에 따른 가중치 변동

뽑기 수가 적을수록 열 가중치 편중이 강화됨:

| 뽑기 수 | 열5 A+B 집중도 | 열2 A+B 출현 |
|---------|-------------|------------|
| 70매 | **47.1%** | **0%** |
| 80매 | 27.7% | 19.2% |

```
뽑기 수 ↓ → 열 가중치 편중 ↑ (특정 열에 몰아넣음)
뽑기 수 ↑ → 열 가중치 균등화 (분산 배치)
```

---

### 규칙 6: 60~80% 구간 = 구조적 "데드존"

```
A등급 60~80% 구간: 11.9% (전 구간 최저)
B등급 40~60% 구간: 15.6% (낮음)
```

뽑기판 3/4 지점 부근이 **양쪽 상위 경품 모두 적은 공백 구간**.
A(앞쪽 편향)와 B(뒤쪽 편향)의 분산 배치가 만들어낸 구조적 사각지대.

## 2. 추정 배치 알고리즘 의사코드

```python
def 뽑기판_생성(총_수, 등급_수량):
    board = [None] * 총_수

    # 1단계: A등급 배치 (앞쪽 편향)
    for each A:
        구간 = 가중치_랜덤([30, 22, 20, 12, 16])
        열 = 가중치_랜덤([19, 19, 26, 7, 29])
        위치 = 구간 내 해당 열의 빈 자리
        if 기존_상위등급과_간격 >= 5:
            board[위치] = 'A'

    # 2단계: B등급 배치 (뒤쪽 편향)
    for each B:
        구간 = 가중치_랜덤([13, 27, 16, 13, 31])
        열 = 가중치_랜덤([20, 24, 9, 20, 27])
        위치 = 구간 내 해당 열의 빈 자리
        if 기존_상위등급과_간격 >= 5:
            board[위치] = 'B'

    # 3단계: C~최하위 등급 순차 배치
    for 등급 in [C, D, E, F, ...]:
        남은 빈 자리에서 랜덤 배치 (약한 가중치 적용)

    return board
```

## 3. 70매 전용 추정 파라미터

70매는 80매 대비 더 극단적인 가중치 사용:

**70매 A등급 추정 열 가중치:**
```
열 1: 25%   열 2: 0%   열 3: 25%   열 4: 12%   열 5: 38%
```

**70매 B등급 추정 열 가중치:**
```
열 1: 11%   열 2: 0%   열 3: 22%   열 4: 22%   열 5: 45%
```

**70매 특이점:**
- 열 2에 A·B 합산 0회 출현 → 열 2가 완전 제외되거나 극도로 낮은 가중치
- 열 5 집중도가 80매 대비 약 1.7배
- 중간 행(행8~11) 완전 공백 → 구간 3~4 가중치가 매우 낮음
